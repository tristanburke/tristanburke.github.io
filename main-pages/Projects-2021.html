Text Editor


This is a basic text editor built from scratch for my CS61B course, titled "Data Structures". The editor includes functions such as word-wrap, type/delete, window-resizing, cursor with clicking and key control, and a basic implementation of undo/redo. The project utilizes many of Java's provided classes and platforms, including JavaFX.

Low Poly


Utilizing Delauney Triangulation methods and common data structures such as heaps, priority queues, and linked lists, this program accepts a user-inputted image and automatically generations a "Low Poly" version. The files are written in Java. Click below to learn more.

Javascript Piano


My second adventure in javascript, I sought to create an easy, clean-looking keyboard. The sounds are not stored locally, and are instead created through Web Audio API. Through this API, I've enabled some customization of the sound, including wave-type and volume. The keyboard itself is completely written in css.
